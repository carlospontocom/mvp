# MVP - Projeto Web

Este é um projeto **MVP (Minimum Viable Product)** desenvolvido com tecnologias modernas de front-end, visando agilidade, performance e escalabilidade.

## 🏦 Descrição

**Agende seus Serviços Bancários com Facilidade**  
Diga adeus às filas e à espera interminável. Nosso sistema de agendamento bancário online permite que você escolha o melhor horário para ser atendido, garantindo praticidade e segurança em cada interação.

Seja para abertura de contas, consultoria financeira, renegociação de dívidas ou atendimento personalizado, oferecemos uma solução moderna que coloca você no controle da sua rotina bancária.


## 🚀 Tecnologias utilizadas

- **[Node.js](https://nodejs.org/)** v22.22.0  
- **[Yarn](https://yarnpkg.com/)** v1.22.19  
- **[Sass](https://sass-lang.com/)** v1.100.0 (compilado com dart2js 3.12.0)  
- **[Vite](https://vitejs.dev/)** v8.0.14
